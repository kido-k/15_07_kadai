時間なく少し仕掛けませんが、起動方法だけ


仮想サーバーとしないとGoogle認証が走らなかったため、
IIS Expressというものを利用して仮想サーバーアドレスを取得
（別途ダウンロードの必要あり）

コマンドプロンプトから

cd c:\Program Files (x86)\IIS Express

　を実行後

iisexpress /path:C:\Users\kidok\OneDrive\ドキュメント\G'S\4回目_20180428\04_15_kadai /port:8080

を実行する。


そうすると、実行するhtmlがあるディレクトリを

http://localhost:8080/

としてブラウザから実行できるので、そこから「main.html」を実行する必要があります。

初回はアカウント作成が必要となります。

実行後、画面下部のマーカ配置を押すと予めセットされていたマーカーが配置されてクリックすると
中身が見れたり、addモードで追加できたり、アカウントの画像やマーカーの画像をドラッグして
セットできたりできます。

簡単ではございますが、以上となります。
